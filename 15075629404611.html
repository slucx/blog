使用Vagrant打造Linux开发环境

2017/10/9 posted in  Unix

基础工作

  1. 安装virtualbox
  2. 安装vagrant
  3. 在virtualbox中安装基本系统

创建box

  1. 创建vagrant用户和用户组，密码为vagrant
  2. root 密码为vagrant
  3. 添加vagrant用户都公钥
mkdir ~/.ssh/
cd ~/.ssh
wget http://github.com/mitchellh/vagrant/raw/master/keys/vagrant
wget http://github.com/mitchellh/vagrant/raw/master/keys/vagrant.pub
mv vagrant.pub authorized_keys
  1. 安装所需的开发工具和ssh服务
  2. sudo apt-get clean
  3. 关闭虚拟机
  4. vagrant package --base debian

使用box

  1. vagrant box add debian /path/of/package.box
  2. vagrant init debian
  3. vagrant up
  4. vagrant ssh
  5. vagrant halt
